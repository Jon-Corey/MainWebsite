Game Tally

A score counter app for board games and card games

About Game Tally

Game Tally is an app for keeping track of the score while playing board games and card games. It's especially helpful for keeping track of health totals in games like Magic: The Gathering or Star Realms.

Game Tally was built from the ground up to be simple and intuitive to use without skimping on features or customization.

It's also completely usable for free (there are no full-screen ads - I hate those). If you'd like to remove the banner ads and get access to more features (more than four counters and more color customizations) then you can get the premium upgrade which goes to support me as I improve this app and create more cool things in the future.
